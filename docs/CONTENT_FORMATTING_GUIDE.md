## 標題層級規範

系統會自動識別以下標題格式，並在對應位置建立分頁：

### Word 內建樣式（推薦）

| Word 樣式 | 用途 | 說明 |
|-----------|------|------|
| 標題 1 (Heading 1) | 主要章節 | 自動觸發分頁，顯示為大標題 |
| 標題 2 (Heading 2) | 次要章節 | 自動觸發分頁，顯示為中標題 |
| 標題 3 (Heading 3) | 小節標題 | 自動觸發分頁，顯示為小標題 |

### Markdown 語法

在純文字中也可使用 Markdown 標記：

```
# 一級標題（觸發分頁）
## 二級標題（觸發分頁）
### 三級標題（觸發分頁）
```

**注意**：`#` 符號必須在行首，後面需接一個空格。

## 章節關鍵字

系統會自動識別以下關鍵字並建立分頁：

### 中式章節編號

```
第一章 人生羅盤
第二節 內在個性
第三篇 事業解析
```

**格式規則**：
- `第X章`、`第X節`、`第X篇`（X 可為數字或中文數字）
- 必須在行首出現

### 方括號標題

```
【人生羅盤】
【內在個性分析】
【重要提醒】
```

**格式規則**：
- 使用全形方括號 `【】`
- 括號內為章節標題
- 建議獨立成行

### 預設章節關鍵字

以下 19 個標準章節名稱會自動觸發分頁：

| 章節名稱 | 章節名稱 | 章節名稱 |
|---------|---------|---------|
| 開場自序 | 使用說明 | 基本資料 |
| 人生羅盤 | 內在個性 | 外在性格 |
| 思考方式 | 行事風格 | 事業解析 |

| 結語大總結 | Bonus：生活方式建議 | Bonus：四時八字軍團 |
| 完成註記 | | |

## 段落格式

### 段落分隔原則

系統會智慧合併短段落，僅在以下情況分段：

1. **句末標點**：遇到 `。`、`！`、`？` 時結束段落
2. **新語意區塊**：標題、引用、列表等新區塊開始時
3. **空行**：連續兩個換行符號

### 建議寫法

✅ **推薦**：
```
這是第一段的內容，包含完整的敘述。這裡繼續說明相關資訊，
直到遇到句號才結束這個段落。

這是第二段的開始，新的主題從這裡展開。
```

❌ **避免**：
```
這是
一個
被
拆
散
的
段落
```

### 文字強調

| 語法 | 效果 | 範例 |
|------|------|------|
| `**文字**` 或 `__文字__` | **粗體** | `**重要提醒**` |
| `*文字*` 或 `_文字_` | *斜體* | `*特別注意*` |

## 表格格式

### Markdown 表格語法

系統支援 Markdown 格式的表格，自動轉換為美觀的 HTML 表格：

```
| 項目 | 說明 | 備註 |
|------|------|------|
| 紫微斗數 | 主星分析 | 重要 |
| 八字 | 五行配置 | 參考 |
| 人類圖 | 能量類型 | 輔助 |
```

### 表格最佳實踐

1. **欄位對齊**：使用 `|` 符號對齊各欄
2. **分隔線**：第二行必須是 `|------|` 格式
3. **避免合併**：系統不支援合併儲存格
4. **控制寬度**：每欄內容盡量簡潔

### 複雜表格範例

```
| 命盤系統 | 主要特質 | 能量走向 | 建議方向 |
|----------|----------|----------|----------|
| 紫微斗數 | 領導力強 | 積極進取 | 適合創業 |
| 八字分析 | 木火旺盛 | 熱情洋溢 | 注意休息 |
| 人類圖 | 顯示者 | 主動發起 | 學會等待 |
```

### 手機與列印注意事項

- 表格在手機上可左右滑動查看
- 列印 PDF 時表格會自動縮放以適應頁面
- 建議表格欄位不超過 5-6 欄

## 圖片插入

### Word 中插入圖片

1. **直接貼上**：在 Word 中直接貼上圖片
2. **插入圖片**：使用「插入 > 圖片」功能

### 圖片建議規格

| 項目 | 建議值 |
|------|--------|
| 格式 | JPG、PNG（推薦 PNG 以保持清晰度）|
| 寬度 | 800-1200 像素 |
| 檔案大小 | 單張不超過 2MB |
| 解析度 | 150-300 DPI |

### 圖片說明文字

在圖片下方添加說明文字時，建議格式：

```
[圖：命盤總覽圖]
或
圖一：紫微斗數命盤
```

### 編輯器中插入圖片

上傳後可在編輯器中：
1. 點擊「插入圖片」按鈕
2. 選擇或上傳圖片
3. 添加圖片說明（可選）
4. 調整圖片位置

## 特殊格式

### 引用區塊

使用 `>` 符號建立引用區塊：

```
> 這是一段重要的引用文字，
> 會以特殊樣式顯示。
```

效果：會顯示為帶有左邊框的引用區塊。

### 手動分頁

在編輯器中可點擊「插入分頁」按鈕，在任意位置建立分頁。

### 列表格式

**有序列表**：
```
1. 第一項
2. 第二項
3. 第三項
```

**無序列表**：
```
- 項目一
- 項目二
- 項目三
```

## 常見問題

### Q1：為什麼我的章節沒有自動分頁？

**檢查項目**：
- 標題是否使用 Word 內建樣式（標題1、標題2）
- 關鍵字是否在行首
- 方括號是否使用全形 `【】`

### Q2：表格顯示異常怎麼辦？

**解決方法**：
- 確保每行的 `|` 數量一致
- 確保有分隔線（第二行 `|---|`）
- 避免在儲存格內換行

### Q3：圖片無法顯示？

**檢查項目**：
- 圖片格式是否為 JPG 或 PNG
- 檔案大小是否過大（建議 < 2MB）
- 嘗試在編輯器中重新插入圖片

### Q4：段落被過度分割？

**解決方法**：
- 減少不必要的換行
- 使用完整句子（以句號結尾）
- 合併相關內容到同一段落

### Q5：PDF 輸出時表格被截斷？

**解決方法**：
- 減少表格欄位數量
- 縮短欄位內容
- 考慮將大表格拆分為多個小表格
