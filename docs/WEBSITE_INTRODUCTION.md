## 🎯 平台定位

**DocShow** 是一個專業的「文件交付與閱讀平台」，專為內容提供者（顧問、創作者、服務方）設計，將 Word 文件轉換為優雅的網頁閱讀版，並提供安全的分享機制。

### 核心價值
- **可控交付**：內容不以被搜尋、被分享為目標，而是精準送達指定讀者
- **專業呈現**：將文件轉化為精美的網頁展示，提升閱讀體驗
- **安全保護**：密碼保護、期限設定、存取追蹤

## 🔐 安全設計

### 資料敏感度分級
| 等級 | 內容類型 | 處理策略 |
|------|----------|----------|
| S0 | 公開文章 | 標準保護 |
| S1 | 商業提案、課程教材 | 加強保護 |
| S2 | 命理報告、客戶資料 | 嚴格保護（預設） |
| S3 | 完整出生資訊、身分資料 | 最高保護 |

### 安全措施
- ✅ Row Level Security (RLS) 全面啟用
- ✅ 分享連結使用不可猜測 Token
- ✅ 權限不足時不顯示任何內容（含標題）
- ✅ 日誌不保存敏感資料
- ✅ 預設不被搜尋引擎收錄
